# Contact Book Web App

A full-stack contact book web application built with Python, Flask, and vanilla JavaScript.

## Features
- Add new contacts with name, phone number, email, and address
- View all saved contacts
- Search contacts by name, phone, email, or address
- Edit existing contact details
- Delete contacts
- Data persists using a JSON file

## Project Structure
```
Contact_book/
    app.py
    contact_book.py
    templates/
        index.html
    static/
        app.js
        style.css
```

## How to Run
1. Install dependencies: `pip install flask flask-cors`
2. Run the app: `python app.py`
3. Open your browser at `http://127.0.0.1:5000`
